## 🚀 ****架构概述****

本站并非基于现成的 Hexo/WordPress 主题模板，而是使用现代前端技术栈**从零构建**的个人知识库。

核心目标是一个**“主次分明”**博客：既能严肃记录技术笔记，又能承载游戏开发的进度展示。

## 🛠️ ****技术栈**** 

为了实现极速加载与高度定制，我选择了以下方案：

- **Astro**: 核心框架。采用“岛屿架构 (Islands Architecture)”，默认输出纯静态 HTML，仅在需要交互时加载 JS，性能极佳。
- **Tailwind CSS**: 样式引擎。通过原子化 CSS 快速构建复杂的 UI 效果。
- **React**: 组件库。用于构建复杂的交互模块（如状态 HUD、工具箱等）。
- **Vercel**: 部署平台。基于 Git 的自动化 CI/CD 部署。
